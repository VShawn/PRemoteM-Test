    Quick_start — Quick Start 项目背景 CONV-19疫情爆发以后，本人不得不在家远程办公。但本人家里的配置是双显示器，一直使用的 mRemoteNG 并不支持 RDP 多屏幕远程，而使用 MSTSC.exe 在会话的启动、切换上又有诸多不爽，于是以此为契机开始打算造个自己满意的 windows 平台远程管理工具，最终就诞生了本项目。 PRemoteM = Personal Remote Manager 重要特性 基于 Tab 页的远程窗体管理 支持 RDP 多屏幕远程 支持 SSH、Telnet 支持HiDPI、UI缩放以及分辨率感知（要求被远程方为 Win10/Win2016 以上系统） 支持快速启动远程会话 多端同步实践 从mRemoteNG快速迁移 安装 确认已安装 .Net framework 4.8 环境。 环境安装包下载 可通过以下任一方式安装软件包 绿色版 Portable package Windows Store 使用 完整说明： 链接 主页面点击 + 按钮添加服务器远程参数。 双击服务器卡片启动远程会话。 Alt+M启动快速搜索，输入关键字并Enter启动远程会话。 在设置→常规中设置语言、程序自启动、最小化启动等。…
